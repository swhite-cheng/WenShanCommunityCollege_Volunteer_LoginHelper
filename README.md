# WenShanCommunityCollege_Volunteer_LoginHelper
login helper the volunteers of WenShan Community College

北市文山社大 志工簽到/簽退 小幫手

1>用手機拍攝 值班當天的 活動驗證碼 (總共四位字元，大寫的英文和數字組成)
2>移動到不擁擠的地方，放大/平移 照片來查看 驗證碼，手動輸入後按下 "複製驗證碼" 到手機的剪貼簿
3>前往 志工系統登入 並依照螢幕輸入 帳號/密碼 (身分證字號 & 民國出生年月日六碼)
4>貼上 "驗證碼" 然後按下 "簽到/簽退"
